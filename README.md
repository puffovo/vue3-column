# vue3-column
基于Vue3、Typescript和Node.js express的一个前后端分离项目，仿知乎专栏--泡芙专栏
# 技术栈
Vue3 + Vuex + Vue-router + TypeScript + ES6 + Bootstap + Express + MySQL
# 项目运行
```
https://github.com/puffovo/vue3-column.git

cd vue3-column

npm run serve

App running at: http://localhost:8080/
```
